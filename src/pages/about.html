About

This template demonstrates how far you can go with zero build tools — pure HTML, CSS, and JavaScript.

Architecture

  • Web Components — Custom Elements + Shadow DOM for encapsulated UI
  • CSS Cascade Layers — Predictable specificity via @layer
  • Design Tokens — All visual decisions in CSS Custom Properties
  • ES Modules — Native import/export, no bundler needed
  • Pub/Sub State — Minimal reactive state management
  • i18n — JSON-based translations with runtime loading
